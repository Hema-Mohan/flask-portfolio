# Simple Flask Portfolio

This is a basic portfolio website built using Flask.

## Features
- Flask routing
- Jinja2 template
- Contact form (POST method)
- Static CSS styling

## How to Run

1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000/
